Project: From JSP to RIA with ICEfaces

With the explosion of Ajax on the web, enterprise Java developers are being asked to deliver more interactive and richer enterprise applications. The cost associated with coming up the Ajax learning curve, plus the cost of migrating your existing JSP infrastructure toward Ajax capabilities can seem insurmountable.

See how ICEfaces can provide the path of least resistance toward migrating your existing JSP-based applications towards Rich Enterprise Applications.

Associated Project Files:

The tutorial references IDE project files that can easily be imported directly to My Eclipse IDE. For other IDEs, the project code would have to be imported manually. Downloadable projects are:

  • jspICEfacesMigration1 - Original JSP
  • jspICEfacesMigration2 - Page-level Migration
  • jspICEfacesMigration3 - Intra-page Migration
