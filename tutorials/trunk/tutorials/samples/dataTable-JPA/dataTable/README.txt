--------------------------------------------------------------------------------
How to Build dataTable-JPA Tutorial 
--------------------------------------------------------------------------------

The dataTable-JPA sample comes with an Ant build script to build the 
application.

Hibernate Core, Hibernate Annotations and Hibernate EntityManager libraries must 
be placed in the folder ./dataTable-JPA/dataTable/lib/ in order for the 
application to successfully compile. These libraries are available at 
http://www.hibernate.org/ .

You will need to download HSQLDB and add it to the lib folder as well.  
http://hsqldb.org/ .
