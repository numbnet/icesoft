Contents

Binary Bundle

You can download ICEfaces as either a binary bundle or a source bundle. The binary bundle comes with the various libraries pre-compiled and the sample applications pre-built to run on Glassfish 3 using JSF 2.0.3. The icefaces directory of the binary bundle contains:
  • compat
    • Libraries and examples related to running ICEfaces 2 in compatibility mode using components from ICEfaces 1.x
    • See the Compatibility topic of the ICEfaces Wiki for more information.
      • lib
        • ICEfaces compatibility library (icefaces-compat.jar) that includes the original suite of ICEfaces 1.x components and code to support running with ICEfaces 2.0.
      • maven
        • Project files to support using ICEfaces compatibility library with Maven.
      • samples
        • ICEfaces 1.x sample applications running in compatibility mode with ICEfaces 2
        • auctionMonitor
        • component-showcase
  • lib
    • ICEfaces (icefaces.jar) and ICEpush (icepush.jar) libraries along with the related compile and runtime dependencies.
  • licenses
    • Summary of legal licenses used by ICEfaces and 3rd party libraries.
  • maven
    • Project files to support using ICEfaces with Maven.
  • samples
    • auction
      • The original ICEfaces Auction Monitor example application ported to use only standard JSF components.
    • basic
      • A simple, pure JSF 2 application that shows how ICEfaces can be easily added to an existing application to provide seamless Ajax support.

Source Bundle

The source bundle contains all the same libraries and applications as the binary bundle but also includes all the source code for the compatibility code and components, the examples, and the core of the ICEfaces framework.
