Demo Use Cases:

1. Click and drag the divider bar between the two areas of content to resize the space they use.

The panel divider component is useful for allowing a user to customize how they want to view a chunk of content. This component is very rich and helps a web application feel more like a traditional desktop application.

The divider orientation can be customized, and the split percentage can be controlled from the server if needed.

The content of each area is populated by filling two facets with the desired components.

Note that the divider component can be embedded within itself, so multiple layered dividers are possible.
