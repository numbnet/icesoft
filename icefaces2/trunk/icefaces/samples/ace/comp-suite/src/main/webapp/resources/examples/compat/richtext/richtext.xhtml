Demo Use Cases:

1. Use the various features of the rich text editor such as bolding text, inserting links, adding lists, etc.

2. Click the "Save" button (icon of a floppy disk) to submit the entered text to the server, where it will display under the "Saved Text String" field at the bottom.

The rich text component wraps the CKEditor and allows users to easily format and customize their input.

All entered text is stored as plain HTML on the server, so developers can redisplay the text, save it as a document, etc. as needed.

The appearance and language of the rich text component can be customized.
