The following tabSet uses the theme default of 'fade' for its transition

The following tabSet overrides the the theme default for its transition. It also demonstrates chaining by using two predefined animations 'highlight' and 'fade'.

 <ice:animation event="transition" name="highlight"/> 
 <ice:animation event="transition" name="fade"/>

The following tabSet uses a fully customised animation, using 'from' and 'to' attributes to mimic blind up/down effect

 <ice:animation event="transition" from="{opacity:1, height:28}" to="{opacity:0, height:1}" easing="backIn"/>

The following demonstrates a customised animation, to mimic pulsate effect, in a standard h:panelGrid

 <ice:animation event="mouseover" to="{opacity:1}" from="{opacity:0}" iterations="3" duration="0.4" easing="easeIn"/>
