Welcome to ICEfaces 2.0.0-Beta2!


This bundle contains software source-code and executables for ICEfaces 2.0.0-Beta2.

All documentation related to ICEfaces 2.0 and the use of this bundle can be found in the ICEfaces Wiki.

Contents

You can download ICEfaces as either a binary bundle or a source bundle.

Binary Bundle

The binary bundle comes with the various libraries pre-compiled and the sample applications pre-built to run on Glassfish 3 using JSF 2.0.3. The binary bundle also includes all the source files for the sample applications, but not the ICEfaces libraries themselves. 

The icefaces directory of the binary bundle contains:
  • ace
    • Examples related to the new ICEfaces Advanced Component Environment (ACE).
    • See the Advanced Components  topic of the ICEfaces Wiki for more information.
      • samples/showcase
        • ICEfaces Advanced Components library showcase sample application.
  • compat
    • Libraries and examples related to using the original ICEfaces Components and/or the ICEfaces 1.x compatibility APIs.
    • See the Compatibility topic of the ICEfaces Wiki for more information.
      • lib
        • ICEfaces compatibility library (icefaces-compat.jar) that includes the original suite of ICEfaces components and a set of ICEfaces 1.x core APIs to ease migration of existing ICEfaces 1.x applications to ICEfaces 2.0.
      • samples
        • ICEfaces 1.x sample applications running on ICEfaces 2 using the original ICEfaces Components and the core compatibility APIs.
        • auctionMonitor
        • component-showcase
  • lib
    • The ICEfaces 2.0 libraries along with related compile-time library dependencies.
      • icefaces.jar - The core ICEfaces framework.
      • icepush.jar - The ICEpush library, provides push functionality to the ICEfaces framework (optional).
      • icefaces-compat.jar - The original ICEfaces Components and 1.x core compatibility APIs (optional).
      • icefaces-ace.jar - The new ICEfaces Advanced Components library (optional).
  • licenses
    • Summary of legal licenses used by ICEfaces and 3rd party libraries.
  • maven2
    • Project files to support using ICEfaces with Maven.
  • samples
    • ICEfaces 2.0 core framework sample applications. Demonstrates using the ICEfaces framework without the ICEfaces Components or Advanced Components.
    • auction
      • An simulated online auction application using only standard JSF components. Demonstrates core benefits of the ICEfaces framework, including the use of push to asynchronously update user's browsers when an auction bid occurs.
    • basic
      • A simple, pure JSF 2 application that shows how ICEfaces can be easily added to an existing application to provide seamless Ajax support.
    • portlets/chat
      • A simple ICEfaces chat portlet that demonstrates using ICEfaces and ICEpush in a portal environment.

Source Bundle

The source bundle contains all the same libraries and applications as the binary bundle but also includes the complete source code for the ICEfaces libraries  and examples. The source bundle does not contain any pre-built ICEfaces libraries or example .wars, etc.
