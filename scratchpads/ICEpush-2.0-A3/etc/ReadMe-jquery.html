Welcome to ICEpush 1.0.0-Alpha3! jQuery Integration Bundle

This bundle contains software source-code and executables for ICEpush 1.0.0-Alpha3 and the jQuery integration and samples.

All documentation related to ICEpush 1.0 and the use of this bundle can be found in the ICEpush Wiki .

    • icechat: A chat demo demonstrating the ICEpush and the jQuery integration

To Build

To build this bundle, run the default target in the root ant script 'build.xml': 

ant build-all -f build.xml
which will build the libraries and all samples.
