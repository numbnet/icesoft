TabSet

Following tabset doesn't specify any animation, but using theme default fade

 <style>
			.default_transition_fade{
				display:block;
			}
</style>

Following tabset using predefine highlight animation, so overriding theme default.

 <ice:animation event="transition" name="highlight"/>

Following tabset using free form animation using from and to attributes to mimic blind up/down effect

 <ice:animation event="transition" from="{opacity:1, height:28}" to="{opacity:0, height:1}" easing="backIn"/>

Following tabset using chaining by defining two predefined animations "fade" and "highlight"

 <ice:animation event="transition" name="highlight"/> 
 <ice:animation event="transition" name="fade"/>

Following h:outputText is inside the h:panelGrid which uses a free from animation to mimic pulsate effect

 <ice:animation event="mouseover" to="{opacity:1}" from="{opacity:0}" iterations="3" duration="0.4" easing="easeIn"/>
